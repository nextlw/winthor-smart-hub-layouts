
# Winthor Smart Hub Layouts

Winthor Smart Hub é um software que faz a ligação entre duas APIs distintas é essencialmente um intermediário que facilita a comunicação entre essas duas APIs. Utilizado em integrações de sistemas heterogêneos, onde as APIs têm diferentes formatos de dados, autenticação, endpoints, entre outros aspectos. No intuito de facilitar sua implantação, layouts com padrões pré-definidos foram .
## Funcionalidades

- Cadastramento de empresas APIs;
- Roteamento de servicos de busca e envio entre as empresas APIs;
- Manipulação de variáveis fixas e dinâmicas;
- Multiplataforma
